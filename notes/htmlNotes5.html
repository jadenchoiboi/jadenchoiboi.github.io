Lesson 5: Semantic Elements


Homepage


A brief overview:

A semantic element are little elements that you put in your line of code that won't show in your real website, but it organizes your title from your content, and in the design phrase, these semantic elements will come into use


Element Header

A container for introductory content

A header element usually contains a logo and one or more heading elements



Element Nav

A container for a set of navigation links



Element Section

This element defines the sections of a document



Element Footer

This is the container for the bottom of the webpage
