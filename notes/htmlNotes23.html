    • Notes 3: Images

  • <img>

    This defines an Image on an HTML PAGE

  • SRC: source

    <img>

    attributes

    • This specifies the URL of an image

  • There are two different ways to use this attribute

    • First option: you can go on google and search for an image. After that, you double click on the image to find the options page. You find the SHORTENED link of the image there and put it in the image command to display your image on your website page.


    Second option: you can save the image file. Save the image containing an HTML code in your finder. It will then pop up in your visual studio code on the very left. Put the link of the image in th image line and there you go.


  • The advantage of using the second method is that if the picture gets deleted off google, it will not show anymore in the code, but the second option will be permanent.
