HTML Notes


A URL = Uniform Resource Locator

  • A URL Speficies a Location on the internet

    • HTML = HyperTextMarkupLanguage

  • This is the language of creating webpages
  • The first line of code should always start off by telling you that it is an HTML file


    • Elements

  • An element is an individual piece of a HTML Document


    • Tags

  • A tag tells you where an element begins and ends


    • <html>

  • The first element in HTML
  • The entire content of an HTML file is inside it



    • There are only 2 elements inside <html>


    <head>

    The head is the place where you put information that will not be visible in your website


    <title>

    This is where you put the text that will be displayed on the top of the page




    <body>

    This is where you store the main content of the website




    <p>

    This makes a paragraph




    <h1>

  • This is used at the header of the document

    • The size of the text decreases as the numbers go to 1-6




    <ul>

  • This creates an empty unordered list



    • <li>

  • This creates one bullet point



    • <ol>

  • This creates an empty ordered list
